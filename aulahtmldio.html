Aula de comunicação Client x Server

Nessas aulas aprendemos a história da web, explicando detalhadamente como os primeiros computadores eram e como surgiram, além de ser citado grandes nomes da computação. Outrossim, é explicado o que são clients no linguajar da web e como funcionam os navegadores e as aplicações web, juntamente com a explicação sobre servidores, como funciona, para que servem e quais são os tipos

Aula de HTML na prática

Nessas aulas, é introduzido como funciona o HTML na prática e quais são os comandos básicos de títulos, parágrafos, listas ordenadas e não ordenada e como funciona a linkagem em sites

Entrega de desafio de projeto

Nessa aula, foi ensinado a como upar o trabalho na plataforma DIO, além de mexer com o básico de github para a entrega desse desafio, que consiste em fazer esse site usando HTML com âncoras, páragrafos e tipos de textos
